# تبسيط واجهة قاعدة المعرفة

حذف زر "إعادة توليد Embeddings" والمنطق المرتبط به، لأن زر "إعادة معالجة الكل" يغطي نفس الوظيفة وزيادة (يعيد التقسيم + يولّد embeddings جديدة).

## التعديلات

### `src/pages/AdminKnowledge.tsx`
- حذف الـ state `regenerating`
- حذف الدالة `handleRegenerateEmbeddings` بالكامل
- حذف زر "إعادة توليد Embeddings" من ترويسة البطاقة (يبقى زر "إعادة معالجة الكل" + زر "رفع مستند")

## النتيجة
- واجهة أبسط وأوضح للمشرف.
- لا تغيير في قاعدة البيانات أو Edge Functions.
- دالة `backfill-embeddings` تبقى موجودة (لم تُستدعَ من الواجهة) ويمكن استخدامها لاحقاً عند الحاجة دون التأثير على المستخدم.
