

# إضافة مربع "مشاريع التخرج السابقة" في القسم الأيسر من صفحة تسجيل الدخول

## التغيير
إضافة عنصر خامس بنفس تصميم المربعات الأربعة الحالية (features grid) في القسم الأيسر (البراندنج) بعنوان "مشاريع التخرج السابقة".

## التنفيذ

### ملف واحد: `src/pages/Login.tsx`
1. إضافة أيقونة `FileText` من `lucide-react` إلى الاستيرادات
2. إضافة عنصر جديد إلى مصفوفة `features`:
   ```
   { icon: FileText, text: "مشاريع التخرج السابقة" }
   ```
3. المربع الجديد سيظهر تلقائياً بنفس التصميم والتنسيق مع المربعات الأخرى (نفس الألوان، الأيقونة، الـ hover، والأنيميشن)
4. بما أن العدد أصبح 5 (فردي) مع grid-cols-2، سيظهر المربع الخامس وحده في صف جديد — سيتم جعله يمتد على عمودين (`col-span-2`) ليكون متمركزاً ومتناسقاً بصرياً
